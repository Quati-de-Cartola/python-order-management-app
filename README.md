# Python Order Management App

A simple desktop application built with Python and Tkinter that allows users to create menu items, place orders and calculate the total price.

## Features

* Add new items to the menu (name and price)
* Display menu items in a list
* Select multiple items and add them to an order
* View the current order
* Calculate the total price of the order
* Simple and clean graphical interface

## Technologies Used

* Python 3
* Tkinter (GUI)
* Object-Oriented Programming (OOP)

## How to Run the Project

1. Clone the repository

```
git clone https://github.com/your-username/python-order-management-app.git
```

2. Open the project folder

```
cd python-order-management-app
```

3. Run the application

```
python main.py
```
